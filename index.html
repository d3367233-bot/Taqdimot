.items() Metodi

dict.items()

Lug'at ichidagi barcha kalit-qiymat juftliklarini ko'rish imkonini beradi.

📋 Kod:

talaba_0 = { 'ism': 'Diyorbek', 'familiya': 'Xudayberdiyev', 'yosh': 20, 'fakultet': 'Axborot texnalogiyalar', 'kurs': 3 } print(talaba_0.items())

🎯 Natija:

Konsol
dict_items([('ism', 'Diyorbek'), ('familiya', 'Xudayberdiyev'), ('yosh', 20), ('fakultet', 'Axborot texnalogiyalar '), ('kurs', 3)])
💡 Maslahat: for tsikli yordamida chaqirish orqali lug'atdagi barcha elementlarni tushunarliroq shaklda ko'rish mumkin.

.items() + for Tsikli

Har bir kalit-qiymat juftligini chiroyli chiqarish:

for kalit, qiymat in talaba_0.items(): print(f"Kalit: {kalit}") print(f"Qiymat: {qiymat} \n")
Natija
Kalit: ism
Qiymat: alijon

Kalit: familiya
Qiymat: shamshiyev

Kalit: yosh
Qiymat: 22

🍎 Amaliy misol:

telefonlar = { 'ali': 'iphone x', 'vali': 'galaxy s9', 'olim': 'mi 10 pro' } for k, q in telefonlar.items(): print(f"{k.title()}ning telefoni {q}")
Natija
Alining telefoni iphone x
Valining telefoni galaxy s9
Olimning telefoni mi 10 pro

.keys() Metodi

dict.keys()

Agar lug'atdagi faqat kalit so'zlarni ko'rish talab qilinsa ishlatiladi.

mahsulotlar = { 'olma': 10000, 'anor': 20000, 'uzum': 40000, 'anjir': 25000 } print(mahsulotlar.keys())
Natija
dict_keys(['olma', 'anor', 'uzum', 'anjir'])
print("Do'kondagi mahsulotlar:") for mahsulot in mahsulotlar.keys(): print(mahsulot.title())
Natija
Do'kondagi mahsulotlar:
Olma
Anor
Uzum
Anjir
Eslatma: for tsiklida .keys() metodini ishlatmasak ham huddi shu natija chiqadi!

bozorlik = ['anor', 'uzum', 'non', 'baliq'] for mahsulot in mahsulotlar: if mahsulot in bozorlik: print(f"{mahsulot.title()} {mahsulotlar[mahsulot]} so'm")

.values() va set()

dict.values()

Lug'atdagi barcha qiymatlarni chiqaradi.

Muammo:

telefonlar = { 'ali': 'iphone x', 'vali': 'galaxy s9', 'hamida': 'galaxy s9', 'umar': 'iphone x' } for tel in telefonlar.values(): print(tel)
iphone x
galaxy s9
galaxy s9 ← Takrorlanmoqda!
iphone x ← Takrorlanmoqda!

Yechim (set()):

for tel in set(telefonlar.values()): print(tel)
iphone x
galaxy s9
✓ Takrorlanish yo'q!
set() haqida: Set - bu tartibsiz, indekslanmagan va takroriy elementlarsiz ma'lumot turi.
