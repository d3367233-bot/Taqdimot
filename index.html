🎯 Lug'at Elementlari Bilan Ishlash

Avvalgi darslarda biz lug'at yaratish va unga kalit bo'yicha murojat qilishni o'rgandik. Ammo real dasturlarda lug'atlar juda katta bo'lishi mumkin va biz barcha kalitlarni yoki qiymatlarni oldindan bilmasligimiz mumkin.

Ushbu qo'llanmada biz quyidagi metodlarni o'rganamiz:

  • 1
    .items() Metodi
    Barcha kalit-qiymat juftliklarini olish uchun
  • 2
    .keys() Metodi
    Faqat kalitlarni olish uchun
  • 3
    .values() Metodi
    Faqat qiymatlarni olish uchun
  • 4
    sorted() Funktsiyasi
    Alifbo bo'yicha tartiblash uchun
  • 5
    set() Funktsiyasi
    Takrorlanishlarni olib tashlash uchun
💡 Nega bu metodlar kerak?

Misol uchun, 1000 ta talaba ma'lumotlari saqlanayotgan lug'atda:

• Faqat ismlarni ko'rish uchun .keys()

• Faqat baholarni ko'rish uchun .values()

• Ism va baholarni birga ko'rish uchun .items()

• Alifbo bo'yicha saralash uchun sorted()

📦 .items() Metodi

Bu metod lug'atdagi barcha kalit-qiymat juftliklarini tuple (o'zgarmas ro'yxat) ko'rinishida qaytaradi.

Oddiy Misol

Python Kod 
# Talaba ma'lumotlari lug'ati
talaba = {
    'ism': 'Diyorbek',
    'familiya': 'Xudayberdiyev',
    'yosh': 20,
    'fakultet': 'Axborot texnalogiyalari'
}

# .items() metodini chaqirish
print(talaba.items())
Natija
dict_items([('ism', 'Diyorbek'), ('familiya', 'Xudayberdiyev'), ('yosh', 20), ('fakultet', 'Axborot texnalogiyalari')])
🧐 Tushuntirish

Natijada har bir juftlik (kalit, qiymat) ko'rinishida tuple ichida saqlangan. Bu o'zgaruvchi strukturani ifodalaydi.

For Tsikli Bilan Ishlash

Oddiy print() chaqirib natija tushunarsiz chiqadi. For tsikli yordamida har bir juftlikni chiroyli ko'rinishda chiqarish mumkin:

Python Kod 
# Ikkita o'zgaruvchi ishlatamiz: kalit va qiymat
for kalit, qiymat in talaba.items():
    print(f"Kalit: {kalit}")
    print(f"Qiymat: {qiymat}\n")
Natija
Kalit: ism Qiymat: Diyorbek Kalit: familiya Qiymat: Xudayberdiyev Kalit: yosh Qiymat: 20 Kalit: fakultet Qiymat: Axborot texnalogiyalari
⚠️ Diqqat!

For tsiklida ikkita o'zgaruvchi ishlatish majburiy. Agar bitta o'zgaruvchi ishlatsangiz, har bir tuple butunligicha shu o'zgaruvchiga yuklanadi.

Amaliy Misol: Telefonlar

Python Kod 
telefonlar = {
    'karamatilo': 'iPhone X',
    'sevara': 'Galaxy S9',
    'dilchehra': 'Mi 10 Pro',
    'diyorbek': 'Nokia 3310'
}

for ism, telefon in telefonlar.items():
    print(f"{ism.title()}ning telefoni {telefon}")
Natija
Karomatiloning telefoni iPhone X Sevarning telefoni Galaxy S9 Dilchehraning telefoni Mi 10 Pro Diyorbekning telefoni Nokia 3310
💡 Foydali Metod

.title() — string metodi bo'lib, so'zning birinchi harfini katta qiladi. Bu foydalanuvchi ismini chiroyli ko'rsatish uchun ishlatildi.

🔑 .keys() Metodi

Bu metod lug'atdagi faqat kalitlarni qaytaradi. Agar sizga qiymatlar emas, faqat kalitlar ro'yxati kerak bo'lsa, bu metoddan foydalanasiz.

Asosiy Qo'llanish

Python Kod 
# Do'kon mahsulotlari
mahsulotlar = {
    'olma': 10000,
    'anor': 20000,
    'uzum': 40000,
    'anjir': 25000,
    'shaftoli': 30000
}

# Faqat kalitlarni olish
print(mahsulotlar.keys())
Natija
dict_keys(['olma', 'anor', 'uzum', 'anjir', 'shaftoli'])

For Tsikli Bilan

Python Kod 
print("Do'kondagi mahsulotlar:")
for mahsulot in mahsulotlar.keys():
    print(f"- {mahsulot.title()}")
Natija
Do'kondagi mahsulotlar: - Olma - Anor - Uzum - Anjir - Shaftoli
💡 Muhim Kashfiyot

Aslida for mahsulot in mahsulotlar: deb yozsangiz ham xuddi shu natija chiqadi! Python avtomatik ravishda faqat kalitlar bo'yicha tsikl qiladi. Bu .keys() metodini ixtiyoriy qiladi, lekin kod o'qilishini yaxshilaydi.

Amaliy Misol: Bozorlik Ro'yxati

Do'kondagi mahsulotlarni bozorlik ro'yxati bilan solishtirish:

Python Kod 
# Narxlar lug'ati
narxlar = {
    'olma': 10000,
    'anor': 20000,
    'uzum': 40000,
    'non': 3000
}

# Bozorlik ro'yxati
bozorlik = ['anor', 'uzum', 'non', 'baliq']

print("Sotib olinadigan mahsulotlar:")
for mahsulot in narxlar:
    if mahsulot in bozorlik:
        print(f"{mahsulot.title()}: {narxlar[mahsulot]} so'm")
Natija
Sotib olinadigan mahsulotlar: Anor: 20000 so'm Uzum: 40000 so'm Non: 3000 so'm
⚠️ Diqqat

'Baliq' bozorlik ro'yxatida bor, lekin narxlar lug'atida yo'q, shuning uchun chiqarilmadi. Bu xatolik emas, shunchaki shart bajarilmadi.

Teskari Tekshirish: Yo'q Mahsulotlarni Topish

Python Kod 
print("\nDo'konda yo'q mahsulotlar:")
for buyum in bozorlik:
    if buyum not in narxlar:
        print(f"- {buyum} (do'konda yo'q)")
Natija
Do'konda yo'q mahsulotlar: - baliq (do'konda yo'q)

💎 .values() Metodi

Bu metod lug'atdagi faqat qiymatlarni qaytaradi. Agar kalitlar emas, faqat ma'lumotlar qiymati kerak bo'lsa ishlatiladi.

Asosiy Qo'llanish

Python Kod 
telefonlar = {
    'karomatilo': 'iPhone X',
    'sevara': 'Galaxy S9',
    'dilchehra': 'Mi 10 Pro',
    'diyorbek': 'Nokia 3310'
}

print(telefonlar.values())
Natija
dict_values(['iPhone X', 'Galaxy S9', 'Mi 10 Pro', 'Nokia 3310'])

Amaliy Misol

Python Kod 
print("Foydalanuvchilar quyidagi telefonlarni ishlatishadi:\n")

for tel in telefonlar.values():
    print(f"📱 {tel}")
Natija
Foydalanuvchilar quyidagi telefonlarni ishlatishadi: 📱 iPhone X 📱 Galaxy S9 📱 Mi 10 Pro 📱 Nokia 3310
⚠️ Muammo!

Agar bir xil qiymatlar ko'p marta takrorlansa, ular ham ko'p marta chiqadi:

Python Kod 
telefonlar = {
    'karomatilo': 'iPhone X',
    'sevara': 'Galaxy S9',
    'dilchehra': 'Galaxy S9',
    'diyor': 'iPhone X',
    'maryam': 'iPhone X'
}

for tel in telefonlar.values():
    print(tel)
Natija (Takrorlanish bilan)
iPhone X Galaxy S9 Galaxy S9 iPhone X iPhone X

set() Yechimi

Takrorlanishlarni olib tashlash uchun set() funksiyasidan foydalanamiz:

Python Kod 
print("Bir xil telefonlarni hisobga olmaganda:\n")

for tel in set(telefonlar.values()):
    print(f"✅ {tel}")
Natija (Takrorlanmas)
Bir xil telefonlarni hisobga olmaganda: ✅ Galaxy S9 ✅ iPhone X

🔄 sorted() Funktsiyasi

Python lug'atlarni kiritilgan tartibda saqlaydi, lekin ko'pincha ma'lumotlarni alifbo bo'yicha tartiblash talab qilinadi. Buning uchun sorted() funksiyasidan foydalanamiz.

sorted() Nima Qiladi?

Bu funksiya har qanday ketma-ketlikni (lug'at, ro'yxat, tuple) olib, elementlarni alifbo yoki o'sish tartibida saralab, yangi ro'yxat qaytaradi.

Metod Natija Tushuntirish
sorted(lugat) Kalitlar ro'yxati Faqat kalitlarni tartiblaydi
sorted(lugat.keys()) Kalitlar ro'yxati Aniqroq yozilishi
sorted(lugat.values()) Qiymatlar ro'yxati Qiymatlarni tartiblaydi
sorted(lugat.items()) Juftliklar ro'yxati Kalit bo'yicha tartiblaydi

Kalitlarni Tartiblash

Python Kod 
mahsulotlar = {
    'olma': 10000,
    'anor': 20000,
    'uzum': 40000,
    'anjir': 25000
}

print("Alifbo bo'yicha mahsulotlar:\n")

for m in sorted(mahsulotlar):
    print(f"• {m.title()}")
Natija
Alifbo bo'yicha mahsulotlar: • Anjir • Anor • Olma • Uzum

Qiymatlarni Tartiblash

Python Kod 
print("Narx bo'yicha (arzondan qimmatga):\n")

for narx in sorted(mahsulotlar.values()):
    print(f"{narx} so'm")
Natija
Narx bo'yicha (arzondan qimmatga): 10000 so'm 20000 so'm 25000 so'm 40000 so'm

Teskari Tartib (Z-A)

Teskari tartibda (kamayish tartibida) chiqarish uchun reverse=True parametridan foydalanamiz:

Python Kod 
for m in sorted(mahsulotlar, reverse=True):
    print(m.title())
Natija (Z-A)
Uzum Olma Anor Anjir

🎯 set() Batafsil

set() — bu faqat takrorlanmas elementlar saqlaydigan ma'lumot turi. Matematikadagi "to'plam" (set) tushunchasiga o'xshaydi.

📋 Set Xususiyatlari
  • ✓
    Takrorlanmas Elementlar
    Set ichida bir xil element faqat bir marta saqlanadi
  • ✓
    Tartibsiz
    Elementlar indeksga ega emas, tartibsiz saqlanadi
  • ✓
    Tez Qidirish
    Hash jadvali tufayli qidirish tez amalga oshiriladi

Set Yaratish

Python Kod 
# Bo'sh set yaratish
bosh_set = set()

# Elementlar bilan set yaratish
meva_set = {'olma', 'anor', 'olma'}  # Olma 2 marta!

print(meva_set)  # {'olma', 'anor'} - takrorlanish olib tashlandi

Ro'yxatni Setga Aylantirish

Bu eng keng tarqalgan usul. Takrorlanishlarni tezda olib tashlash uchun:

Python Kod 
telefonlar = ['iPhone', 'Samsung', 'iPhone', 'Nokia', 'Samsung']

# Setga aylantirish
unique_telefonlar = set(telefonlar)

print(unique_telefonlar)
print(f"Jami: {len(telefonlar)} ta, Unique: {len(unique_telefonlar)} ta")
Natija
{'Samsung', 'iPhone', 'Nokia'} Jami: 5 ta, Unique: 3 ta
💡 Amaliy Maslahat

Lug'at qiymatlarida takrorlanishlar ko'p bo'lsa (masalan, 1000 ta foydalanuvchidan faqat 10 xil telefon modeli), avval set() bilan unique modellarni ajratib oling, keyin ularni sanang yoki tahlil qiling.

Topshiriq 1
Python Izohli Lug'ati

Python izohli lug'atini yarating va lug'atga kamida 10 ta so'z qo'shing. Lug'atdagi har bir kalit va qiymatni for tsikli yordamida, alifbo ketma-ketligida chiroyli qilib konsolga chiqaring.

Yechim 
# Python atamalar lug'ati
python_lugat = {
    'boolean': 'Mantiqiy qiymat (True yoki False)',
    'float': 'O\'nlik kasr son',
    'for': 'Takrorlash (tsikl) operatori',
    'if': 'Shart operatori',
    'integer': 'Butun son',
    'list': 'O\'zgaruvchan ro\'yxat',
    'loop': 'Takrorlash tsikli',
    'string': 'Matn (tekst) ma\'lumot turi',
    'tuple': 'O\'zgarmas ro\'yxat',
    'variable': 'O\'zgaruvchi'
}

print("📚 PYTHON LUG'ATI\n")
print("=" * 40)

# Alifbo bo'yicha tartiblab chiqarish
for atama, izoh in sorted(python_lugat.items()):
    print(f"\n🔹 {atama.upper()}")
    print(f"   {izoh}")
Topshiriq 2
Davlatlar va Poytaxtlar

Davlatlar va ularning poytaxtlari lug'atini tuzing. Avval lug'atdagi davlatlarni, keyin poytaxtlarni alohida-alohida, alifbo ketma-ketligida konsolga chiqaring.

Yechim 
davlatlar = {
    'aqsh': 'vashington',
    'germaniya': 'berlin',
    'rossiya': 'moskva',
    'uzbekistan': 'toshkent',
    'xitoy': 'pekin'
}

print("🌍 DAVLATLAR:\n")
for d in sorted(davlatlar.keys()):
    print(f"• {d.upper()}")

print("\n🏛️ POYTAXTLAR:\n")
for p in sorted(davlatlar.values()):
    print(f"• {p.title()}")
Topshiriq 3
Davlatni Qidirish

Foydalanuvchidan istalgan davlatni kiritishni so'rang va shu davlatning poytaxtini konsolga chiqaring. Agar foydalanuvchi lug'atda yo'q davlatni kiritsa, "Bizda bunday ma'lumot yo'q" degan xabarni chiqaring.

Yechim 
davlatlar = {
    'aqsh': 'vashington',
    'germaniya': 'berlin',
    'rossiya': 'moskva',
    'uzbekistan': 'toshkent'
}

davlat = input("Davlat nomini kiriting: ").lower()

if davlat in davlatlar:
    print(f"\n✅ {davlat.upper()}ning poytaxti {davlatlar[davlat].title()}")
else:
    print("\n❌ Bizda bunday ma'lumot yo'q")
Topshiriq 4
Restoran Menyusi

Restoran menusi lug'atini tuzing. Foydalanuvchidan 3 ta ovqat buyurtma berishni so'rang. Foydalanuvchi kiritgan taomlarni menu bilan solishtiring, agar taom menuda bo'lsa narhini ko'rsating, aks holda "bizda bunday taom yo'q" degan xabarni chiqaring.

Yechim 
menu = {
    'osh': 25000,
    'manti': 8000,
    'somsa': 6000,
    'shashlik': 15000,
    'lagmon': 22000
}

print("🍽️ 3 ta taom buyurtma bering:\n")

for n in range(3):
    taom = input(f"{n+1}-taom: ").lower()
    
    if taom in menu:
        print(f"   ✅ {taom.title()}: {menu[taom]} so'm\n")
    else:
        print(f"   ❌ Bizda {taom} yo'q\n")
