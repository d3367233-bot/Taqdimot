💡 Eslatma

Lug'atlar katta yoki kichik bo'lishi mumkin. Ba'zida barcha kalitlarni yoki qiymatlarni oldindan bilmasligimiz mumkin.

📋 .items() Metodi

Barcha kalit-qiymat juftliklarini ko'rish uchun ishlatiladi.

PYTHON 
talaba = {
    'ism': 'Ali',
    'yosh': 22,
    'kurs': 4
}

for kalit, qiymat in talaba.items():
    print(f"{kalit}: {qiymat}")
Natija
ism: Ali
yosh: 22
kurs: 4
📱 Telefonlar Misoli
PYTHON 
telefonlar = {
    'ali': 'iPhone',
    'vali': 'Samsung'
}

for ism, tel in telefonlar.items():
    print(f"{ism.title()}ning telefoni {tel}")
Natija
Alining telefoni iPhone
Valining telefoni Samsung
🔑 .keys() Metodi

Faqat kalit so'zlarni olish uchun ishlatiladi.

PYTHON 
mahsulotlar = {
    'olma': 10000,
    'anor': 20000,
    'uzum': 40000
}

print(mahsulotlar.keys())

for m in mahsulotlar.keys():
    print(m.title())
Natija
dict_keys(['olma', 'anor', 'uzum'])
Olma
Anor
Uzum
💡 For tsiklida .keys() yozmasangiz ham natija bir xil!
🛒 Bozorlik Misoli
PYTHON 
bozorlik = ['anor', 'non']

for m in mahsulotlar:
    if m in bozorlik:
        print(f"{m}: {mahsulotlar[m]} so'm")
💎 .values() Metodi

Faqat qiymatlarni olish uchun ishlatiladi.

PYTHON 
telefonlar = {
    'ali': 'iPhone',
    'vali': 'Samsung',
    'olim': 'iPhone'
}

for tel in telefonlar.values():
    print(tel)
Natija
iPhone
Samsung
iPhone
🔄 set() - Takrorlanmas

Bir xil qiymatlar faqat bir marta chiqishi uchun:

PYTHON 
for tel in set(telefonlar.values()):
    print(tel)
Natija
iPhone
Samsung
🔄 sorted() Funktsiyasi

Lug'at elementlarini alifbo bo'yicha tartiblab chiqarish.

PYTHON 
mahsulotlar = {
    'olma': 10000,
    'anor': 20000,
    'uzum': 40000
}

# Alifbo bo'yicha kalitlar
for m in sorted(mahsulotlar):
    print(m)
Natija
anor
olma
uzum
💡 sorted(lugat) - kalitlarni tartiblaydi
💡 sorted(lugat.values()) - qiymatlarni tartiblaydi
